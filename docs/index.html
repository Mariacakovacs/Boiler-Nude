    mixins

    center

    @mixin center() { ... }

    Description

    Center a Block Element Horizontally or Vertically This mixin makes your life when centering block elements a cake you just need to define true or false depending on the centering you need The parent element should have the position set to relative

    Parameters

    None.

    Example

    foo

    .foo {
	@include center;

    &.horizontal {
		@include center(true, false);
		
	}

	&.vertical {
	@include center (false, true);
   }
	
}
